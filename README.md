## GitHub Pages での公開

1. GitHub のリポジトリ設定で **Pages** を開きます。
2. **Build and deployment** の **Source** で、公開したいブランチを選びます。
3. ルート (`/`) を公開対象にすると、`index.html` がトップページとして配信されます。

公開後は、トップページから各 HTML へ相対リンクで移動できます。
